# Responsive Design

Diseño responsive mobile-first.

## Breakpoints

| Prefix | Tamaño |
|--------|--------|
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |

## Patrones

```
Grid:       grid-cols-1 md:grid-cols-2 lg:grid-cols-3
Ocultar:    hidden md:flex
Texto:      text-sm md:text-base lg:text-lg
Padding:    p-4 md:p-6 lg:p-8
Navigation: hamburguesa en móvil, horizontal en desktop
```

## Enfoque
- Mobile-first: estilos base para móvil
- Añadir breakpoints progresivamente
- Probar en cada breakpoint
