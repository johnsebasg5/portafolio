# Portfolio Sections

Estilos específicos para secciones de portafolio.

## Estructura por Sección

| Sección | Estilo |
|---------|--------|
| Hero/Landing | Centrado vertical, bg-gradient, CTA prominente |
| About | 2 columnas, skills con badges |
| Projects | Grid de cards con hover effects |
| Contact | Formulario centrado, links sociales |
| Topbar | Sticky, backdrop-blur, responsive nav |
| Footer | Simple, links, copyright |

## Container Base
```
max-w-6xl mx-auto px-4 py-12
```

## Paleta de Colores
- **Primary**: blue
- **Secondary**: purple
- **Neutral**: slate
- **Success**: green
- **Warning**: yellow
- **Error**: red
