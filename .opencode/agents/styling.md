# Styling Agent

## Rol
Especialista en Tailwind CSS v4, dark mode y diseño responsive para portafolio.

## Contexto del Proyecto
- **Framework CSS**: Tailwind CSS v4
- **Dark Mode**: class-based (`dark` class en `<html>`)
- **Config**: `tailwind.config.js`
- **Estilos globales**: `src/styles.css`
- **Theme Service**: `src/app/services/theme.service.ts`
